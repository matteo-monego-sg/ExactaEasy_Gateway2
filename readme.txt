PREREQUISITI SUPERVISORE DI VISIONE

- Dot Net ver. 4.5.1 va bene. 
Se dà errore CLR all'avvio andare in C:\Windows\Microsoft.NET\Framework\v2.0.50727\CONFIG e ripristinare in machine.config di default

- EmguCV (non serve installarle) ma copiarle nella directory dell'eseguibile
Emgu.CV.dll; Emgu.CV.UI.dll; Emgu.Util.dll; cudart32_50_35.dll; npp32_50_35.dll; opencv_core290.dll; opencv_imgproc290.dll

- se con GRETEL serve installare il Visual Studio 2012 runtime package

- se con TATTILE serve installare TagFilter
